# Bridge Adapter

"Bridge Adapter" & "Bridge Adapter Dialog" stories describe using of the Bridge Adapter component.

To use it properly, create a project ID at [WalletConnect](https://cloud.walletconnect.com).

The best way to see the demo is by pasting the project ID via Storybook's controls and navigating to the separate component page: [iframe.html?args=&id=bridge-adapter--default&viewMode=story](iframe.html?args=&id=bridge-adapter--default&viewMode=story).
